CONOCIENDO.

La presente explicación será una breve introducción a otras opciones de jugar los videojuegos a las ya existentes

RETROARCH

una aplicación que sirve de base para instalar en ella diferentes emuladores y ejecutar con ellos las ROMs. Por lo tanto, para poder utilizar los juegos de determinada consola no vale con sólo descargar RetroArch, sino que después tendrás que cargar en la aplicación el emulador correspondiente.

RETROARCH

Es un frontend que nos permite emular consolas de distintas generaciones.

            
                - Abra el navegador https://www.retroarch.com/.
                - Nos dirigimos a Descargas y seleccionamos la plataforma.

BATOCERA

Es una distribucion Linux que permite reutilizar equipos antiguos dependiendo de las caracteristicas de este permite emular distintos dispositivos

BATOCERA

            
                - Abra el navegador https://batocera-linux.xorhub.com/.
                - Nos dirigimos a Descargas y seleccionamos la plataforma.
                - Instalacion: https://batocera-linux.xorhub.com/#install.
                - Compatibilidad: https://batocera-linux.xorhub.com/#systems.

X LINK KAI

XLink Kai es una red global de juegos - que reúne a XBox /360, PlayStation 2 / 3, Gamecube y los usuarios de PSP, en una comunidad integrada. Se trata de un software que se ejecuta en su PC, Macintosh o Linux que le permite reproducir el sistema de enlace habilitado de juegos en línea GRATIS.

X LINK KAI

            
                - Abra el navegador https://es.teamxlink.co.uk/.

                - Instalacion: https://batocera-linux.xorhub.com/#install.
                - Compatibilidad: https://batocera-linux.xorhub.com/#systems.
